# Overview

In this project, I built a complete Snake Game using Java. This project helped me practice structuring a full application, managing multiple classes, and creating a simple user interface using Swing.

The program is not just the game itself, but a small application that includes a main menu (lobby), navigation between screens, and a score system. The user can start the game, view saved scores, or exit the application.

The purpose of this project is to demonstrate my ability to build a functional Java application, handle user interaction, and manage data such as saving and displaying scores.

[Software Demo Video](https://youtu.be/uCNvU_G_H84)

# Development Environment

For this project, I used:

* Visual Studio Code

* Java JDK
* Java Swing (for the graphical interface)

The programming language used was Java.

# Features

* Main menu (Lobby) with options:

  * Play the game
  * View scores
  * Exit the application

* Snake Game built with Java Swing

* Keyboard controls (arrow keys + ESC to return to menu)

* Score system that saves player name and points

* Scores stored in a file and sorted in descending order

* Separate screen to display ranking
